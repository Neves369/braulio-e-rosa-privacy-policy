Política de Privacidade - Braulio e Rosa App

Tipos de dados coletados

Entre os tipos de dados pessoais que o Braulio e Rosa App recolhe, por si só ou por meio de terceiros, estão: cookies, dados de uso, localização geográfica, tokens para envio de notificações e informações fornecidas durante a criação de contas de usuários.

Outros dados pessoais recolhidos podem ser descritos em outras seções desta política de privacidade ou por meio de textos explicativos específicos apresentados no contexto da coleta de dados.

Os dados pessoais podem ser livremente fornecidos pelo usuário, ou coletados automaticamente durante o uso do Braulio e Rosa App.

Qualquer uso de cookies ou de outras ferramentas de rastreamento pelo aplicativo, ou pelos proprietários dos serviços terceirizados utilizados, salvo indicação em contrário, serve para identificar os usuários e lembrar suas preferências, com o único propósito de fornecer os serviços solicitados.

O não fornecimento de determinados dados pode impossibilitar o funcionamento correto do Braulio e Rosa App.

O usuário assume a responsabilidade pelos dados pessoais de terceiros publicados ou compartilhados por meio do aplicativo e confirma que tem o consentimento da parte para fornecer os dados ao controlador.

Modo e local de processamento dos dados

Método de processamento

O controlador de dados processa os dados dos usuários de maneira adequada e adota as medidas de segurança necessárias para impedir o acesso não autorizado, divulgação, alteração ou destruição não autorizada dos dados.

O processamento dos dados é realizado por meio de computadores e/ou ferramentas de TI, seguindo procedimentos organizacionais e meios estritamente relacionados com as finalidades indicadas. Além do controlador de dados, em alguns casos, os dados podem ser acessados por pessoas envolvidas na operação do aplicativo (como administração, vendas, marketing, suporte técnico) ou por fornecedores externos (como serviços de hospedagem, provedores de TI e comunicação), nomeados, quando necessário, como processadores de dados. A lista atualizada dessas partes pode ser solicitada ao controlador de dados a qualquer momento.

Local

Os dados são processados nos escritórios operacionais do controlador de dados e em quaisquer outros lugares onde as partes envolvidas no processamento estejam localizadas. Para mais informações, entre em contato com o controlador de dados.

Período de conservação

Os dados são mantidos pelo tempo necessário para fornecer os serviços solicitados ou para as finalidades descritas nesta política, e o usuário pode solicitar ao controlador de dados que os suspenda ou remova.

Uso dos dados coletados

Os dados coletados são utilizados para permitir que o Braulio e Rosa App forneça seus serviços, incluindo: criar contas de usuário, enviar notificações e interagir com redes sociais e plataformas externas.

Os dados pessoais utilizados para cada finalidade estão descritos nas seções específicas desta política de privacidade.

Informações detalhadas sobre o processamento de dados pessoais

Criar conta de usuário

O Braulio e Rosa App permite a criação de contas de usuário, onde os dados fornecidos (como nome, e-mail e senha) são armazenados de maneira segura. Estes dados são necessários para autenticação e acesso aos serviços oferecidos pelo aplicativo.

Enviar notificações ao usuário

O Braulio e Rosa App pode enviar notificações ao usuário. O push token do dispositivo será adicionado a uma lista de tokens que podem receber notificações do aplicativo. Essa lista não contém outras informações do usuário, garantindo o anonimato.

Coleta de localização geográfica

O Braulio e Rosa App pode coletar dados de localização geográfica do usuário para melhorar a experiência e fornecer serviços baseados em localização. A coleta da localização é feita de maneira automática quando o aplicativo está em uso, respeitando as permissões concedidas pelo usuário em seu dispositivo.

Informações adicionais sobre a coleta e processamento de dados

Ação jurídica

Os dados pessoais dos usuários podem ser utilizados para fins jurídicos pelo controlador de dados em juízo ou nas etapas conducentes a uma possível ação legal decorrente do uso indevido do aplicativo ou dos serviços relacionados.

O usuário declara estar ciente de que o controlador de dados pode ser obrigado a revelar dados pessoais mediante solicitação das autoridades governamentais.

Logs do sistema e manutenção

Para fins de operação e manutenção, o Braulio e Rosa App e quaisquer serviços de terceiros podem coletar arquivos que registram a interação com o aplicativo (logs do sistema) ou usar outros dados pessoais (como endereço IP).

Os direitos dos usuários

Os usuários têm o direito de consultar o controlador de dados para saber se os seus dados pessoais foram armazenados, verificar sua exatidão, solicitar a atualização, remoção ou transformação em formato anônimo, e bloquear o processamento de dados em violação à lei. Os pedidos devem ser enviados ao controlador de dados utilizando as informações de contato fornecidas no final deste documento.

Mudanças nesta política de privacidade

O controlador de dados se reserva o direito de fazer alterações nesta política de privacidade a qualquer momento, mediante comunicação aos usuários nesta página. É recomendável consultar esta página regularmente. Se o usuário não concordar com as alterações, deve interromper o uso do Braulio e Rosa App e pode solicitar que o controlador de dados apague suas informações pessoais.

Informações legais

Aviso aos usuários europeus: esta declaração de privacidade foi elaborada em conformidade com as obrigações do Regulamento Geral de Proteção de Dados (GDPR).
